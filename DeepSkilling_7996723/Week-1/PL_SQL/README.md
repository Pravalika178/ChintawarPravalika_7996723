# PL/SQL Banking Exercises

This workspace contains Oracle PL/SQL scripts for a banking exercise set covering:
- Control Structures
- Error Handling
- Stored Procedures
- Functions
- Triggers
- Cursors
- Packages

## Files
- `sql/setup.sql` &mdash; schema creation and sample data inserts
- `sql/exercise1_control_structures.sql` &mdash; control structure examples
- `sql/exercise2_error_handling.sql` &mdash; exception handling examples
- `sql/exercise3_stored_procedures.sql` &mdash; stored procedures for banking operations
- `sql/exercise4_functions.sql` &mdash; reusable functions
- `sql/exercise5_triggers.sql` &mdash; triggers for audit and data integrity
- `sql/exercise6_cursors.sql` &mdash; cursor-based processing
- `sql/exercise7_packages.sql` &mdash; packages grouping related logic

## Usage
Run these scripts in an Oracle SQL environment (SQL*Plus, SQL Developer, or any Oracle-compatible client). Execute `sql/setup.sql` first to create the schema and load sample data.
